# RiftHub

> Community-ranked discussions and discovery platform for Africa's tech space

Rifthub is a platform where founders, developers, students, researchers, and technology enthusiasts can discover, discuss, and rank stories shaping the African technology ecosystem. The platform provides a focused space for meaningful conversations around startups, artificial intelligence (AI), cloud computing, open-source projects, venture capital, digital infrastructure, and emerging innovations across Africa.


## Why I Build RiftHub?

The African tech ecosystem is growing rapidly, yet discussions are often fragmented across social media platforms and messaging groups. The idea was to bring these conversations into a single community-driven platformm where users can participate in thoughtful discussions relevant to tech, discover emerging startups, and keep up with the trends across the tech space.

## Getting Started

**Prerequisities**

- Node.js 18+ 
- Python 3.12+ 
- uv
- PostgreSQL

**Installation**

Clone the repository:

```bash
git clone https://github.com/Bh3ky/the-beacon.git
cd the-beacon
```

Frontend Setup:

```bash
npm install

npm run web:dev
```

Backend Setup:

```bash
cd apps/api

uv sync

npm run api:dev
```
